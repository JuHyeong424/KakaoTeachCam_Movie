깃허브 페이지
https://juhyeong424.github.io/KakaoTeachCam_Movie/

API_KEY는 현재까지 배운 내용에서 .env를 사용하지 못하여 public으로 두었습니다.


1. Home 클릭시
   - 기본으로 제공하는 페이지의 1페이지로 이동한다.
2. 원하는 단어를 검색
   - 해당 단어를 포함한 결과가 나온다.
3. 페이지 이동
   - 기본에서는 최대 500페이지까지 검색할 수 있다.
   - 검색된 페이지에서는 데이터에서 나온 max_page까지 나올 수 있다.
   - 원하는 페이지를 검색할 수 있다.
